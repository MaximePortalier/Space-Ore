## <span style="color:#17D7B6;">Installation</span>

To get started with <strong>Space Ore</strong>, follow these simple steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/MaximePortalier/Space-Ore.git
   ```

2. Install packages:
   ```bash
   npm install
   ```
3. Launch project:
   ```bash
   npm run dev
   ```
